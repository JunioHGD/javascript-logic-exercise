# Object Methods

Esta pasta reúne exercícios sobre métodos e recursos de manipulação de objetos no JavaScript.

## Conteúdos praticados

- `Object.keys()`
- `Object.values()`
- `Object.entries()`
- `Object.assign()`
- `Object.hasOwn()`
- Spread operator
- Destructuring
- Acesso, atualização e reorganização de propriedades

## Objetivo

Praticar formas de acessar, transformar, combinar e reorganizar dados armazenados em objetos.
