# Array Methods

Esta pasta reúne exercícios focados nos principais métodos de array do JavaScript.

## Conteúdos praticados

- `forEach`
- `map`
- `filter`
- `find`
- `some`
- `every`
- `reduce`
- Combinação de métodos

## Objetivo

Entender como cada método funciona, o que ele retorna e em quais situações pode ser aplicado para resolver problemas de manipulação de dados.
