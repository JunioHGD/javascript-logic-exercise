# 03 - Data Manipulation Methods

Esta pasta reúne exercícios sobre métodos e recursos de manipulação de dados no JavaScript.

## Subcategorias

- `array-methods`: métodos de array.
- `string-methods`: métodos de string.
- `object-methods`: métodos e recursos para manipular objetos.

## Objetivo

Praticar formas de transformar, filtrar, buscar, validar e reorganizar dados usando recursos nativos da linguagem.
