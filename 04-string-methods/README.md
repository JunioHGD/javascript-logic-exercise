# 04 - String Methods

Esta pasta reúne exercícios sobre métodos de manipulação de strings no JavaScript.

## Conteúdos praticados

- `includes`
- `split`
- `replace`
- `trim`
- Transformação de textos
- Validação simples de entradas textuais

## Objetivo

Praticar formas de buscar, dividir, substituir, limpar e transformar textos usando métodos nativos de strings.
