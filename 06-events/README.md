# 06 - Events

Esta pasta reúne exercícios sobre eventos no JavaScript.

## Conteúdos praticados

- Eventos de clique
- Eventos de formulário
- Eventos de teclado
- `addEventListener`
- Objeto de evento
- Interações básicas do usuário com a página

## Objetivo

Entender como capturar ações do usuário e responder a essas interações usando JavaScript.
