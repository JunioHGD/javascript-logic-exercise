# 05 - DOM Basics

Esta pasta reúne exercícios introdutórios sobre manipulação do DOM com JavaScript.

## Conteúdos praticados

- Seleção de elementos
- Alteração de conteúdo
- Alteração de atributos
- Manipulação de classes
- Criação e remoção de elementos
- Renderização simples de dados na tela

## Objetivo

Praticar a conexão entre JavaScript e elementos HTML, preparando a base para projetos interativos com interface.
