# 01 - Fundamentals

Esta pasta reúne exercícios sobre os fundamentos do JavaScript.

## Conteúdos praticados

- Variáveis
- Tipos de dados
- Operadores
- Condicionais
- Funções
- Estruturas básicas de lógica

## Objetivo

Construir uma base sólida nos recursos essenciais da linguagem antes de avançar para manipulação de dados, DOM, eventos e desafios mais completos.
