# 02 - Data Structures

Esta pasta reúne exercícios sobre estruturas de dados básicas no JavaScript.

## Conteúdos praticados

- Arrays
- Objetos
- Listas de dados
- Propriedades e valores
- Acesso e atualização de informações
- Organização de dados para resolver problemas

## Objetivo

Praticar formas de armazenar, acessar e manipular dados usando estruturas fundamentais da linguagem.
